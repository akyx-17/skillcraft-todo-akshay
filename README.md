# SkillCraft To-Do Web App - Akshay

A simple To-Do web application for SkillCraft Task 04.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark as completed
- Set date & time
- Organize tasks into lists
- Saved in localStorage

## How to Run (optional)
1. Install Node.js from https://nodejs.org/
2. Run:
npm install
npm run dev
